# PD Webové Mesto – Konceptuálny redizajn agropodniku

Moderná, rýchla a plne responzívna webová prezentácia poľnohospodárskeho družstva.
---

## O projekte

Tento projekt je kompletne novou, modernizovanou verziou webovej stránky, ktorá v minulosti úspešne fungovala ako oficiálna prezentácia reálneho poľnohospodárskeho družstva (PD Melčice-Lieskové). 

Pôvodný koncept som od základu pretransformoval, dal mu novú identitu „PD Webové-Mesto“ a kód obliekol do špičkového digitálneho šatu. Z dôvodu organizačných zmien vo firme a ochrany údajov boli informácie na webe upravené. Web dnes slúži ako plne funkčná, optimalizovaná ukážka mojej práce, rýchleho načítania a intuitívneho ovládania na mobile.

## Hlavné funkcie a vlastnosti

* **Kompletná lokalizácia:** Web podporuje plnohodnotný prepínač medzi slovenskou a anglickou verziou (multijazyčnosť).
* **Dynamický filter jedálneho lístka:** JavaScriptové filtrovanie produktov/ponuky podľa kategórií bez nutnosti prenačítavať stránku.
* **Mobile-First Design & UX:** Web je navrhnutý s dôrazom na pohodlné ovládanie na smartfónoch a prístupnosť (A11y/ARIA atribúty).
* **Optimalizácia výkonu:** Použitie moderných formátov obrázkov (`.webp`) pre bleskové načítanie stránky.
* **Interaktívne prvky:** Vlastné rezervačné modálne okná s okamžitou spätnou väzbou pre používateľa.
